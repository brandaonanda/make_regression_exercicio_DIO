# 📈 make_regression - Exercício DIO

Atividade desenvolvida durante o **bootcamp da DIO/TOTVS**, com foco em fundamentos de **Machine Learning**.

---

## 🎯 Objetivo

Utilizar a função `make_regression` para:

* Gerar uma base de dados sintética
* Simular um problema de regressão
* Visualizar os dados através de um gráfico

---

## 🧠 Conceitos Aplicados

* Geração de dados artificiais
* Regressão linear
* Visualização de dados

---

## 🛠️ Tecnologias Utilizadas

* Python
* Scikit-learn
* Matplotlib

---

## ⚙️ Como Executar

1. Instale as dependências:

```bash
pip install scikit-learn matplotlib
```

2. Execute o script Python:

```bash
python script.py
```

---

## 📚 Contexto

Exercício prático para entender como funcionam dados de entrada em modelos de regressão e como eles podem ser visualizados para análise.
